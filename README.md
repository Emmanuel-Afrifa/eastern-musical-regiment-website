# Eastern Musical Regiment Website

## Overview
Official repository for the Eastern Musical Regiment website. Uses Nextjs and tailwind css for the frontend application and Strapi for content management.

## Repository Structure
- frontend/ – Next.js frontend application
- cms/ – Strapi CMS backend

## Tech Stack
- Frontend: Next.js, React, Tailwind, Jest, Framer Motion
- Backend: Strapi

## Getting Started
- Clone the repo
    - If you use `HTTPS`, use:
    ```bash
    git clone https://github.com/Emmanuel-Afrifa/eastern-musical-regiment-website.git
    ```

    - If you use `SSH`, use:
    ```bash
    git clone git@github.com:Emmanuel-Afrifa/eastern-musical-regiment-website.git
    ```

- Install dependencies
   - Navigate into each directory (`frontend/` and `cms/`) and run:
    ```bash
    npm install
    ```
- Start the frontend
    ```bash
    cd frontend
    npm run dev
    ```

- Start CMS
    ```bash
    cd cms
    npm run develop
    ```
